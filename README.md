云研 ELN 功能说明书

项目名称：云研（Yunyan ELN）  
项目形态：单文件 Web 应用（yunyan.html）  
技术栈：React 18 + ReactDOM 18（UMD）+ Babel + Tailwind CSS（CDN）  
定位：面向研发团队的电子实验记录本（ELN）SaaS 原型系统


---

1. 项目目标

云研用于实现研发场景下的实验数字化记录与协作管理，覆盖实验创建、记录检索、项目管理、文档管理、权限治理、审计追踪与支持服务等核心流程，支持客户演示、交互验收与毕业设计答辩展示。

3. 核心功能说明

3.1 导航与交互框架
- 左侧树形导航支持分组与折叠。
- 支持快捷键：N 新建实验，G 记录列表，A 数据分析，Ctrl+B 侧栏折叠。
- 统一深色极简风格，强调高信息密度与操作效率。
  
3.2 新手引导动画（重点）
- 首次进入“工作台看板”自动触发引导。
- 引导时长不低于 30 秒。
- 流程：点击新建实验 → 自动输入标题 → 上传图表/数据 → 进入分析页。
- 支持手动重播引导。
  
3.3 实验记录能力
- 支持创建实验（标题、项目、模板、富文本正文、附件上传）。
- 富文本区域支持基础编辑交互。
- 拖拽上传区支持文件加入列表与进度反馈。
- 记录列表支持查询浏览，点击进入只读预览。
- 只读预览支持审阅、审批提交、导出（并写入审计日志）。
  
3.4 数据分析能力
- 提供分析维度切换（趋势、对照、异常等）。
- 指标切换（峰值、均值、斜率、方差）。
- 展示洞察卡片与异常点列表。
- 支持生成报告与导出动作入口。
  
3.5 项目与资产管理
- 项目管理支持项目列表展示与新增。
- 项目详情展示里程碑、关联实验、模块跳转。
- 文档资料库支持检索、预览/下载、入库操作。
- SOP 模板库支持模板查看、应用、新建。
  
3.6 组织治理能力
- 团队成员管理：成员列表、邀请、重邀、移除。
- 权限配置：RBAC 矩阵展示与交互切换。
- 操作日志：按时间、操作者、动作、对象、结果追踪。
  
3.7 个人与安全
- 站内信通知：消息列表、详情、已读/归档。
- 个人偏好：默认项目、默认模板、自动保存等。
- 账号安全：2FA、登录告警、设备会话管理。
- 存储分析：容量占比、分类统计、优化建议。
- 帮助工单：知识库 + 工单创建与列表追踪。

4. 数据与状态设计

4.1 统一数据仓
系统采用统一前端状态模型，核心数据包括：
- 项目（projects）
- 实验记录（records）
- 文档（docs）
- SOP（sops）
- 成员（members）
- 日志（logs）
- 消息（messages）
- 偏好（prefs）
- 安全（security）
- 存储（storage）
- 工单（tickets）
  
4.2 本地持久化
- 使用 localStorage 保存业务数据（键：yunyan_app_data_v1）。
- 刷新页面后可恢复状态。
- 首次引导状态单独持久化控制。
  

---

5. 审计与可追溯

- 系统内关键操作统一记录日志（如创建记录、审批、导出、成员变更、工单创建等）。
- 日志字段包含：时间、操作者、动作、对象、结果。
- 满足毕业设计对“过程可追踪”的展示要求。
  

---

6. 非功能说明

- 单文件部署：无需打包，双击 HTML 可运行。
- 前端原型性质：当前无后端接口，数据保存在浏览器本地。
- 可扩展：后续可平滑接入后端 API、数据库、对象存储与权限服务。
